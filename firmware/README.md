# Firmware

`main.c` is the MicroBlaze host driver: it reconfigures the pixel
clock via `clk_wiz`, uploads a linear-to-sRGB LUT and framebuffer clears
through the CPU-store path, and issues hand-assembled GPU instructions
(`0xNNNNNNNN` words written to the command FIFO at `B+16`) to rasterize
textured/shaded triangles from a small software 3D pipeline (fixed-point
Q16.16 matrix math, clip-space vertex packing, perspective divide).

Depends on Xilinx BSP headers (`platform.h`, `xil_io.h`) generated by
Vitis for the target hardware platform, not included here.

See [`../fpga_assembler`](../fpga_assembler) for the assembler that turns
the mnemonics referenced in the comments (e.g. `cpu_store`, `barrier`,
`swg`) into the raw instruction words this file pokes directly.
